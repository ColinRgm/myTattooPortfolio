Questionnaire

Question 1 :

Expliquer la différence entre une requête HTTP synchrone et asynchrone.

Une requête synchrone est une requête où le navigateur est bloqué durant l’attente de la réponse.
Exemple : lorsque l’on clique sur un lien hypertexte.

Une requête asynchrone est une requête où le navigateur peut s’occuper sur une autre tâche durant l’attente de la réponse.
Exemple : Recharge partielle d’une page.

Question 3 :

Écrire vos recommandations (best practice) pour programmer un site web en HTML / CSS.

Mes conseils seraient d’utiliser des commentaires pour pouvoir ce retrouver plus simplement dans son code, être organisé dans son code pour pouvoir facilement le relire et le comprendre et faire les choses dans l’ordre, je dirais qu’il faudrait aussi faire un zooming pour avoir un aperçu de ce que l’on veut et avoir une sorte de marche à suivre.

HTML : Utiliser les commentaires pour chaque section de son code (navbar, titre, contenu, etc).

CSS : Tout comme pour le HTML utiliser des commentaire pour différencier les parties et essayer de faire dans l’ordre d’affichage (commencer par le haut).
